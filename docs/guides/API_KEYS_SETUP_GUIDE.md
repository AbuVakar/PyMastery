## 🔧 **How to Add API Keys to PyMastery**

### **Step 1: Open AI Mentor Page**
```
1. Go to your PyMastery application
2. Navigate to AI Mentor page
3. Look for left sidebar
```

### **Step 2: Add API Key**
```
1. Find "🔑 API Keys (Optional)" section
2. Enter your API key in the relevant field:
   - OpenAI API Key → "OpenAI API Key" field
   - Google Gemini → "Google Gemini API Key" field  
   - Groq → "Groq API Key" field
   - Hugging Face → "Hugging Face Token" field
3. Keys are automatically saved to browser
```

### **Step 3: Select Model & Start Chatting**
```
1. Choose your preferred AI model from the list
2. Start typing your Python questions
3. Get real AI responses!
```

## 🔒 **Security & Best Practices**

### **🛡️ Keep Your Keys Safe:**

- ❌ **Don't commit** to git repositories
- ✅ **Use environment variables** in production
- ✅ **Store securely** in browser only
